📦 파일: iBuild-Screen-Calculator-Final.html

🎉 완성 기능:
✅ 비밀번호 로그인 (Screen10000**)
✅ Google Sheets 온라인 저장
✅ 어디서든 인터넷으로 접속 가능
✅ 프린트 최적화 (Letter 사이즈 한 장)
✅ 검색 & 정렬
✅ 설치장소 정보 포함

═══════════════════════════════════════════════════

🚀 사용 방법:

1. 파일 열기
   - iBuild-Screen-Calculator-Final.html 더블클릭

2. 로그인
   - 비밀번호: Screen10000**
   - (대소문자 정확히, 별표 2개)

3. 계산 & 저장
   - 가로/세로 입력
   - 고객 정보 입력
   - 📊 계산하기
   - 💾 저장하기

4. 저장 내역 확인
   - 📂 저장 내역 클릭
   - 검색, 정렬, 불러오기 가능

5. 프린트
   - Ctrl+P (Windows) 또는 Cmd+P (Mac)
   - Letter 사이즈, 세로 방향
   - 한 장에 모든 내용 출력

═══════════════════════════════════════════════════

🌐 Google Sheets 확인:

1. https://sheets.google.com 접속
2. "iBuild 방충망 데이터" 찾기
3. 저장된 모든 데이터 확인!

컬럼 구조:
A: 저장시간
B: 타입 (편개형/양개형)
C: 가로(inch)
D: 세로(inch)
E: 고객명
F: 날짜
G: 주소
H: 설치장소
I: 이메일
J: 전화번호
K: 프레임색상
L: 망색상
M: 계산결과 (JSON)

═══════════════════════════════════════════════════

✨ 주요 기능:

📊 계산:
- 편개형 & 양개형
- 인치/mm 단위 변환
- 실시간 계산

💾 저장:
- 온라인 자동 저장

📋 정렬:

- 크기순

🖨️ 프린트:
- Letter 사이즈 최적화
- 한 장 출력
- 회사 정보 포함

🔧 문제 해결:

Q: 로그인이 안 돼요
A: 비밀번호를 정확히 입력하세요
   Screen10000** (대소문자 구분, 별표 2개)

A: 인터넷 연결을 확인하세요
   F12 → Console에서 에러 확인

Q: 데이터가 안 보여요
A: 📂 저장 내역 버튼을 클릭하세요
   Google Sheets에서도 직접 확인 가능
